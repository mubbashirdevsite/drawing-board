# 🎨 Drawing Board App

A modern **Canvas Drawing Application** built using HTML, CSS, and Vanilla JavaScript.

---

## 🚀 Features

- Freehand drawing (mouse + touch)
- Pencil tool
- Eraser tool
- Color picker
- Brush size control
- Opacity control
- Undo / Redo
- Clear canvas
- Save as PNG
- Responsive canvas
- Keyboard-ready structure
- Smooth drawing experience

---

## 🛠️ Technologies Used

- HTML5 Canvas API
- CSS3
- JavaScript (Vanilla)
- Touch & Mouse Events
